# Personal PDF Toolbox – Final Feature Checklist

## ✅ Essential Features (Core)

### 1. PDF Conversion
- PDF → Word (DOCX)
- PDF → Excel (XLSX)
- PDF → PowerPoint (PPTX)
- Word/Excel/PowerPoint → PDF
- PDF → Images (JPG, PNG)
- Images → PDF
- PDF → Text (via OCR for scanned PDFs)

### 2. PDF Editing & Manipulation
- Merge multiple PDFs into one
- Split a PDF into multiple files
- Rotate pages (90°, 180°, 270°)
- Extract specific pages
- Rearrange pages
- Add watermarks
- Add annotations or simple text overlays

### 3. PDF Security
- Protect PDFs with passwords
- Unlock password-protected PDFs
- Remove permissions/restrictions

### 4. PDF Optimization
- Compress PDFs by reducing image resolution
- Optimize file size without losing content
- Convert scanned PDFs into searchable text (OCR)

### 5. User Experience
- Drag-and-drop file upload
- Multi-file batch processing
- Immediate download of converted or edited files
- Temporary local storage with auto-cleanup
- Lightweight web interface or terminal usage

### 6. Developer-Friendly Features
- Fully open-source, free tools (no external paid API)
- Local execution
- Modular architecture (conversion, editing, security, optimization separated)
- Async processing for large files (optional)
- Easy extension for new formats or custom processing
